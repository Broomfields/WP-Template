# World Building

This is where everything to do with the world itself lives — separate from the stories set within it.

The [wiki](wiki/_overview.md) is the main event: the canonical reference for everything that exists in this world. Characters, locations, cultures, magic, history — if it's a fact about the world, it belongs in there.

[Assets](assets/_overview.md) sits alongside it for images, Excalidraw sketches, and anything else visual I want to reference from the wiki or elsewhere.
