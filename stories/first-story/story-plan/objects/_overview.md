# Objects

Story-significant items that play a narrative role beyond what their wiki entry describes. Not every item in the wiki needs an entry here — only those with real story weight: a MacGuffin, a symbolic object, something that changes hands or transforms through the narrative, or an item that carries thematic significance in this particular story.

If an item has a wiki entry, that entry holds the canonical description of what it is and its place in the world. This file holds what it *means in this story*.

## What an Entry Represents

One entry per story-significant item. Covers its narrative role, symbolic weight, how it moves through the story (who holds it, loses it, finds it, destroys it), and the key scenes that involve it.

## How Entries Relate to the Rest of the System

- Entries link to the item's wiki entry (if one exists) via `wiki_entry`
- Entries reference the arc in which the object is introduced
- If an object has no wiki entry — it exists only in the story, not the broader world — leave `wiki_entry` blank

## Linking Convention

Use an Obsidian wikilink in `wiki_entry` if the item has a wiki entry: `wiki_entry: "[[object-slug]]"`. If it's a story-only item with no world-encyclopedia presence, just leave it blank.
